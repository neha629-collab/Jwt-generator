FF JWT API

Guest UID + Password, Access Token, and EAT Token based JWT generator

Endpoints

- GET /api/public/token?guest_uid=<uid>&guest_password=<pw>
- GET /api/public/token?access_token=<token>&open_id=<open_id>
- GET /api/public/token?eat_token=<eat>
If EAT conversion endpoint is unsupported by upstream token content, this may fail.
